Embedding Media
& Online Resources

HTML4 <object>

Until recently, browsers didn't have any built-in ways to handle video or sound, so they required external plug-ins play media files. Those plugins are no longer available and that method doesn't work with the latest versions of HTML. This is an example of the way it used to be done:

"Ordinary People" by William Shatner

HTML5 <audio>

HTML5 has a built-in <audio> element and all web browsers support .mp3 files, so it's pretty easy to embed audio in that format.

"I Can't Get Behind That" by William Shatner & Henry Rollins

HTML5 <video>

The good news is that HTML5 has a built-in <video> element, so no browser plug-ins are required. The bad new is that the various browsers don't agree on what video formats should be supported.

That means you should offer your video in multiple file formats (.mp4, .webm, .ogv) to make sure it can play in all browsers. This can be done with the <source> element.

Embed YouTube Videos

YouTube players will be visible in the VSCode preview window, but will give an error if you try to play them there. You'll need to open the page in an actual browser to play it.

You may find some embedded YouTube videos don't work and say "Video Unavailable." The most common cause is when the video contains copyrighted music, then for legal reasons YouTube blocks it from playing when embedded.

Optional Video Attributes:

controls
If this attribute is present, the browser will buttons to control the video. Without this attribute the user won't be able to play/pause the video or control the volume.
autoplay
With this attribute the video automatically begins to play back as soon as it loads. This is annoying and probably shouldn't be used.
loop
Automatically loops back to the beginning when it reaches the end of the video.
poster
A URL for an image file that will show before you hit play. Without this attribute it will show the first frame of video.
height
Sets the height of the video player.
width
Sets the width of the video player.
preload
Tells the browser to start downloading the video file before the user hits the play button.
